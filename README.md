# Enhanced Inventory Management Application

This project was enhanced as part of my CS499 capstone to demonstrate advanced software development skills.

This repository contains an enhanced version of my original Android inventory management application developed during my coursework in the Computer Science program.

---

## Purpose

The goal of this enhanced version is to demonstrate my growth as a software developer by improving the application across three key areas:

- Software Design and Engineering  
- Algorithms and Data Structures  
- Databases and Security  

---

## Key Enhancements

### Software Design and Engineering
- Refactored code using a more structured architecture (MVC/MVVM)
- Improved code organization and modularity
- Added input validation and error handling
- Enhanced user interface and usability

---

### Algorithms and Data Structures
- Implemented search functionality for inventory items
- Added sorting features (by name, quantity, price)
- Improved data handling efficiency

---

### Databases and Security
- Added user authentication system
- Implemented secure password storage using hashing
- Introduced salted hashing to protect against security vulnerabilities
- Improved database schema with user and inventory tables
